### 3.2 GET /search
- HTTP 200 + data.results array voi: userId, name, picture, level, friendStatus
- friendStatus = "none" khi chua co quan he
- Khong thay chinh minh trong ket qua
- q="a" (1 ky tu) → 400 QUERY_TOO_SHORT
- khong co q → 400 VALIDATION_ERROR
- **BUG:** "tuan"/"cuong"/"trinh" → 0 results. "Tuan" (co dau) → tim thay. Search dung LIKE/ilike khong co unaccent()

### 3.3 POST /add-friend

- DB: 1 row `(A→B, pending)` chinh xac
- Search sau khi gui → friendStatus = "request_sent"
- Gui lan 2 → 409 REQUEST_ALREADY_SENT
- Tu add chinh minh → 400 CANNOT_ADD_SELF
